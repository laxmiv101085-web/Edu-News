# Icon Files

The PWA requires icon files for proper installation and display. Currently, placeholder text files are in place.

## Required Icons

Replace the following placeholder files with actual PNG images:

1. **`apps/web/public/icon-192x192.png`**
   - Size: 192x192 pixels
   - Format: PNG
   - Purpose: App icon for mobile devices

2. **`apps/web/public/icon-512x512.png`**
   - Size: 512x512 pixels
   - Format: PNG
   - Purpose: App icon for larger displays and splash screens

3. **`apps/web/public/badge-72x72.png`**
   - Size: 72x72 pixels
   - Format: PNG
   - Purpose: Badge icon for notifications

## Creating Icons

You can create these icons using:
- Design tools (Figma, Adobe Illustrator, etc.)
- Online icon generators
- Image editing software (GIMP, Photoshop, etc.)

## Icon Design Tips

- Use a simple, recognizable design
- Ensure good contrast for visibility
- Test on both light and dark backgrounds
- Follow platform guidelines (iOS, Android, Web)

## Quick Solution

For quick testing, you can use a simple colored square or download free icons from:
- [Flaticon](https://www.flaticon.com)
- [Icons8](https://icons8.com)
- [Material Icons](https://fonts.google.com/icons)

Make sure to resize them to the exact dimensions specified above.
