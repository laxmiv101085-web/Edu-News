## Educational App Web Client

### Getting Started

```bash
npm install
npm run dev
```

Environment variables:

```
NEXT_PUBLIC_API_URL=http://localhost:4000
```

### Quality Gates

- `npm run lint`
- `npm run test`
- `npm run storybook`
- `npm run storybook:build`
- `npm run test:e2e`

### Storybook

Run `npm run storybook` and open `http://localhost:6006` to explore the design system tokens and component stories.

### Tests

- Unit tests: `npm run test`
- Watch mode: `npm run test:watch`
- E2E smoke test (Playwright): `npm run test:e2e`
